## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/AlbericoJr/Cesta_Frutas
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
``

Inicie o servidor

  npm start
